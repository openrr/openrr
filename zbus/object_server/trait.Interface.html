Expand description

The trait is used to dispatch messages to an interface instance.

This trait should be treated as unstable API and compatibility may break in minor version bumps. Because of this and other reasons, it is not recommended to manually implement this trait. The crate::dbus_interface macro implements it for you.

If you have an advanced use case where dbus_interface is inadequate, consider using crate::MessageStream or crate::blocking::MessageIterator instead.

fn spawn_tasks_for_methods(&self) -> bool

Whether each method call will be handled from a different spawned task.
