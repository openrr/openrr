A convenience macro to ergonomically define an item depending on a large number of #[cfg] parameters. Structured like match statement, the first matching branch is the item that gets emitted.

match_cfg
The macro provided by this crate, match_cfg, is similar to the if/elif C preprocessor directives and allows defining a cascade of #[cfg] cases, emitting the implementation which matches first.
