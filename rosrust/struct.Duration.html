Struct Duration

Source 
pub struct Duration {
    pub sec: i32,
    pub nsec: i32,
}
Expand description

ROS representation of duration, with nanosecond precision

Fields§

§sec: i32

Number of seconds. Negative for negative durations.

§nsec: i32

Number of nanoseconds inside the current second. Negative for negative durations.
