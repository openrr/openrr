pub fn read(&self) -> Result<u64>

Reads the value from the file descriptor.

  • If EFD_SEMAPHORE was not specified and the eventfd counter has a nonzero value, then this function returns an u64 containing that value, and the counterโs value is reset to zero.

  • If EFD_SEMAPHORE was specified and the eventfd counter has a nonzero value, then this function returns an u64 containing the value 1, and the counterโs value is decremented by 1.

  • If the eventfd counter is zero at the time of this call, then the call either blocks until the counter becomes nonzero (at which time, this function proceeds as described above) or fails with the error EAGAIN if the file descriptor has been made nonblocking with EFD_NONBLOCK.
