Struct BitReaderRtl

Source 
pub struct BitReaderRtl<'a> { /* private fields */ }
Expand description

BitReaderRtl reads bits from least-significant to most-significant from any &[u8].

Stated another way, if N-bits are read from a BitReaderRtl then bit 0, the first bit read, is the least-significant bit, and bit N-1, the last bit read, is the most-significant.
