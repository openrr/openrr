Expand description

Describes the discriminant of an enum,and its valid values.

§Safety

This must be an enum with a #[repr(C)] or #[repr(SomeInteFgerType)] attribute.

The Discriminant associated type must correspond to the type of this enum’s discriminant.

The DISCRIMINANTS associated constant must be the values of this enum’s discriminants.
