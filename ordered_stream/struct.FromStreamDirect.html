Struct FromStreamDirect

pub struct FromStreamDirect<S, F> { /* private fields */ }
Expand description

An OrderedStream wrapper around a Stream.

This does not use any future or past knowledge of elements, and so is suitable if the stream rarely or never blocks. Prefer using FromStream if you plan to filter or join this stream and want other streams to be able to make progress while this one blocks.
